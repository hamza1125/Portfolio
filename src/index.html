ROBOFRIENDS

This is a simple React.js framework application using a few APIs to create a stateful web application. Utilizing the APIs from JSONPlaceholder for user information and RoboHash for user images. Style for the page makes use of the Tachyons framework.

This was the first application that I’ve built using React.js, and as such is simple but responsive. Starting from the create-react-app package, the code was written with JavaScript/JSX. The search and title stays on the screen regardless of size, and the number of cards across will change based on width. The app dynamically draws only the cards that are applicable to the content in the search box.
